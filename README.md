# orion360-sdk-pro-examples-ios
Examples for Orion360 SDK (Pro) for iOS

## Google Ima Example

### Setup

- Clone the repository
- Install pods:
```
cd GoogleImaTwoIsolatedPlayers
pod install --repo-update
```
- Launch GoogleImaTwoIsolatedPlayers.xcworkspace
